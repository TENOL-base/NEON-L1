# NEON L-1

## Лабораторный стенд для изучения архитектуры процессоров

**NEON L-1** — это учебный 8-битный процессорный стенд, предназначенный для наглядного изучения принципов работы вычислительных систем.

Стенд позволяет наблюдать выполнение программ **на уровне сигналов, регистров и передачи данных** в реальном времени.

Проект ориентирован на использование в:

* высших учебных заведениях
* лабораторных работах по архитектуре ЭВМ
* курсах по цифровой электронике и микропроцессорной технике

---

## Назначение

NEON L-1 разработан как инструмент, который позволяет:

* изучать цикл выполнения команд
* понимать взаимодействие модулей процессора
* наблюдать передачу данных внутри системы
* работать с памятью и внешними устройствами на низком уровне

В отличие от классических решений, стенд делает внутренние процессы **доступными для прямого наблюдения и анализа**.

---

## Наблюдение работы процессора

Стенд оснащён системой визуализации на базе более чем 200 светодиодов, отображающих:

* состояние регистров
* передачу данных
* управляющие сигналы
* активность модулей
* тактовый процесс

Это позволяет:

* отслеживать выполнение программы шаг за шагом
* анализировать поведение системы
* выявлять ошибки на уровне логики

---

## Основные характеристики

* 8-битная архитектура
* модульная организация системы
* визуализация внутренних процессов
* универсальные двусторонние порты
* программируемая память

---

## Работа с внешними устройствами

NEON L-1 использует универсальные 8-битные порты, которые позволяют подключать внешние устройства через аппаратные модули.

Поддерживаются реализации интерфейсов:

* VGA
* PS/2
* UART
* I2C
* SPI
* USB (через внешний контроллер)

Такой подход позволяет изучать не только работу процессора, но и принципы взаимодействия с периферией.

---

## Программная экосистема

Стенд сопровождается набором инструментов:

### Программатор

Аппаратный интерфейс для:

* загрузки программ
* управления процессором
* наблюдения состояния

### BitProg

Программное обеспечение, обеспечивающее:

* разработку программ
* компиляцию
* взаимодействие с процессором

### TenBit

Язык программирования, разработанный для данной архитектуры и ориентированный на обучение.

## С чего начать

Если вы впервые работаете с NEON L-1:

1. Ознакомьтесь с документацией процессора
   `docs/CPU-NEON-L1-RU.pdf`

2. Установите программное обеспечение
   → `software/app/`

3. Изучите работу программатора
   `software/programmer/`

4. Ознакомьтесь с языком TenBit
   `software/tenbit/`

5. Запустите примеры программ
   `examples/`
